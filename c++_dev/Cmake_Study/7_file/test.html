    Photoshop을 사용하여 컬러 이미지를 흑백으로 신속하게 변환하는 방법을 알아보십시오.

    앱에서 도움말을 확인하세요!

    Photoshop 앱에서 컬러 이미지를 흑백으로 변환하는 관련 툴, 도움말 문서 및 튜토리얼을 살펴보세요.

    컬러 이미지를 흑백으로 변환하는 방법

    이미지의 색상 및 음영 또는 색조가 뚜렷하고 선명한 경우 컬러 포토그래피는 독특하고 선명하여 복잡한 세부사항도 표현할 수 있습니다. 흑백 사진은 색상으로 인해 주의가 분산되지 않고 이미지에서 피사체와 텍스처에 초점을 맞추려는 경우에 더 좋습니다. 흑백 효과를 사용하면 이미지에 드라마 효과를 연출할 수 있습니다.

    Photoshop의 흑백 조정 레이어를 사용하여 컬러 이미지를 흑백으로 쉽게 변환하려면 아래 빠른 단계를 따르십시오.

    1. 흑백 조정 레이어를 사용하여 이미지를 흑백으로 변환

      흑백 조정 레이어에 액세스하려면 다음 중 하나를 수행합니다.

      • 창 > 조정을 선택합니다. 열리는 조정 패널에서 [흑백] 아이콘()을 클릭합니다.
      • 레이어 > 새 조정 레이어 > 흑백을 선택합니다. 새 레이어 대화 상자에서 조정 레이어의 이름을 입력하고 확인을 클릭합니다.

      Photoshop은 이미지에 기본 회색 음영 변환을 적용합니다.

    2. (선택 사항) [속성] 패널에서 결과를 세부적으로 조정합니다.

      사전 설정: 미리 정의된 회색 음영 혼합 또는 이전에 저장한 사용자 정의 혼합을 선택합니다. 사용자 정의 혼합을 저장하려면 속성 메뉴() 아이콘을 클릭하고 [흑백 사전 설정 저장]을 선택합니다.

      자동: 이미지의 색상 값을 기반으로 회색 음영 혼합을 설정합니다. 자동 혼합을 사용하면 주로 우수한 결과를 얻을 수 있으며, 색상 슬라이더를 사용하여 회색 값을 세밀하게 조정하는 시작점으로 사용할 수도 있습니다.

      색조: 색상 톤을 적용하려면 색조를 활성화합니다. 색상 견본을 클릭하여 [색상 피커]를 열고 색조 색상을 선택합니다.

      색상 슬라이더: 이미지의 특정 색상의 회색 톤을 조정합니다. 슬라이더를 왼쪽으로 드래그하여 이미지의 원래 색상에 해당하는 회색 톤을 어둡게 하거나 오른쪽으로 드래그하여 밝게 합니다. 

      • 캔버스에서 직접 색상을 조정하려면 이미지의 조정 도구()를 사용합니다. 이미지를 클릭하고 왼쪽(어둡게) 또는 오른쪽(밝게)으로 드래그하여 해당 위치의 해당 색상에 대한 색상 슬라이더를 수정합니다.
      • 조정 기본값을 재설정하려면 [재설정]() 아이콘을 클릭합니다.

    3. 결과 저장 및 공유
      흑백 변환을 통해 만족스러운 결과를 얻었다면 Adobe Photoshop 커뮤니티 및 소셜 미디어에 결과를 저장하고 공유해 보십시오. Adobe Photoshop을 태그하고 이미지에 #blackandwhiteshop 해시태그를 지정하는 것을 잊지 마십시오.

      저장 및 공유 워크플로우에 대한 자세한 내용은 다음을 참조하십시오.

      • 파일 내보내기
      • 생성한 내용 빠르게 공유
